 Project Overview
This project demonstrates the implementation and analysis of cryptographic systems ranging from classical ciphers to modern Substitution-Permutation Networks (SPN). It focuses on key cryptographic principles such as confusion, diffusion, non-linearity, and avalanche effect.

The final implementation includes a Python-based SPN model inspired by modern encryption standards such as AES.

---

Objectives
- Implement classical encryption algorithms
- Study stream and block cipher concepts
- Build a Substitution-Permutation Network (SPN)
- Demonstrate S-Box substitution and permutation
- Analyze non-linearity in cryptographic systems
- Compare SPN and Feistel structures
- Demonstrate avalanche effect
